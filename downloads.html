Linux

Linux is it! Your distro has probably already packaged git-cola for you.
If not, please file a bug against your distribution ;-)
arch → pacman -S git-cola
debian → apt-get install git-cola
fedora → yum install git-cola
gentoo → emerge cola
ubuntu → apt-get install git-cola
More? Contact the list if you've packaged git-cola for other distributions.

Mac OS X

You can install git-cola using Brew. Brew installs git-cola's dependencies.
You can install newer versions of git-cola after you've installed it using brew.
  • brew install git-cola
  • After installing git-cola using Brew you can install the git-cola application bundle

Windows

Install the latest stable Python 2.6 binary packages listed below
  • git
  • python
  • pyqt4
  • git-cola
